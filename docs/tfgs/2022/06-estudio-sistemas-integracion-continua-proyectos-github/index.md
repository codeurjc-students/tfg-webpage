## Resumen

de integración continua que podemos encontrar en repositorios de código abier-

de este estudio es obtener información relativa a estos sistemas de integración
continua, como por ejemplo, conocer cuál es la herramienta predominante en ca-
da plataforma, si se suelen combinar más de una herramienta de este tipo en
